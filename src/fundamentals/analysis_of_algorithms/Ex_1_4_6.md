    for(int i = 0; i < n; i++)

log N * (N + N/2 + N/4 + ... + N/N)

log N * N(1 + 1/2 + 1/4 + ... + 1/N) <- [Geometric series](https://en.wikipedia.org/wiki/Geometric_series)

log N * N(2) = O(N log(N))
